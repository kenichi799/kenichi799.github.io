如何解锁 Xperia 设备的 bootloader

请注意，解锁您Xperia设备的bootloader将会清除设备上的所有数据，并且可能会使您的保修失效。此过程通常建议高级用户进行。请谨慎操作，风险自负。

重要注意事项 (请仔细阅读):

  • 数据丢失: 解锁bootloader将擦除设备上的所有数据。在继续之前，请务必将您的所有重要数据（照片、视频、联系人等）备份到您的电脑或外部存储设备。

  • DRM密钥: 解锁bootloader还会清除存储在TA分区中的DRM密钥。这可能会影响某些索尼专有功能，例如相机后期处理、色彩配置文件、降噪、X-Reality视频增强、DSEE HX、ClearAudio+以及对HD Netflix的Widevine L1支持。即使您重新锁定bootloader，这些密钥也无法恢复。

  • 保修: 解锁bootloader可能会使您的设备保修失效。

  • 重新锁定Bootloader: 只有在所有内容都是原厂状态（原始固件）的情况下，您才能重新锁定bootloader。否则，您可能会面临数据丢失甚至设备变砖的风险。

准备工作:

  1. 检查是否允许解锁Bootloader:

    • 在您的手机上打开拨号器。

    • 输入 *#*#7378423#*#* 以访问服务菜单。

    • 点击 Service info > Configuration > Rooting Status。

    • 如果 Bootloader unlock allowed 显示为 Yes，则您可以继续下一步。如果显示为 No 或状态缺失，则您的设备无法使用此方法解锁。

  2. 下载并安装USB驱动程序:

    • 访问索尼开发者官方网站下载适用于您特定Xperia型号的USB驱动程序: https://developer.sony.com/open-source/aosp-on-xperia-open-devices/downloads/drivers

    • 在您的电脑上安装驱动程序。

  3. 下载并解压ADB和Fastboot工具:

    • 从Google下载平台工具（包含ADB和Fastboot）: https://dl.google.com/android/repository/platform-tools-latest-windows.zip

    • 将ZIP文件的内容解压到您电脑上的一个文件夹中（例如：C:\platform-tools）。

  4. 配置ADB环境变量 (可选但推荐):

    • 这允许您从任何命令提示符窗口运行ADB命令。

    • Windows 10 及以上版本: 按下 Windows 键，输入“环境变量”，然后选择“编辑系统环境变量”。

    • 在“系统属性”窗口中，点击“环境变量...”按钮。

    • 在“系统变量”部分，找到名为“Path”的变量并选中它。

    • 点击“编辑...”按钮。

    • 在“编辑环境变量”窗口中，点击“新建”，然后添加您解压平台工具的目录的绝对路径（例如：C:\platform-tools）。

    • 点击所有打开的窗口中的“确定”以保存更改。

  5. 在您的Xperia设备上启用USB调试和OEM解锁:

    • 转到 设置 > 系统 > 关于手机。

    • 连续快速点击 版本号 （通常点击7次），直到看到“您现在是开发者！”的消息。

    • 返回到 设置 > 系统 > 高级 > 开发者选项。

    • 启用 USB调试 选项。

    • 启用 OEM解锁 选项（如果可用）。

  6. 将您的Xperia设备连接到电脑:

    • 使用USB数据线将您的手机连接到电脑。

    • 在您的手机上，您可能会看到一个提示询问是否允许USB调试。勾选“始终允许来自此计算机”，然后点击“允许”。

  7. 打开命令提示符或终端:

    • 打开命令提示符窗口（在Windows上）或终端（在macOS或Linux上）。

    • 使用 cd 命令导航到您解压平台工具的目录。例如，如果您解压到 C:\platform-tools，则输入 cd C:\platform-tools 并按Enter键。

  8. 进入Fastboot模式:

    • 进入Fastboot模式的方法可能因您的Xperia型号而略有不同。以下是一种常见方法：

    • 完全关闭您的手机。

    • 按住 音量增大 按钮。

    • 在按住音量增大按钮的同时，通过USB数据线将您的手机连接到电脑。

    • 继续按住音量增大按钮，直到您的手机进入Fastboot模式。您可能会看到黑屏或Fastboot标志。

  9. 使用Fastboot命令解锁Bootloader:

    • 在命令提示符或终端窗口中，输入以下命令并按Enter键: 

      fastboot oem unlock

    • 注意: 在某些较新的Xperia设备上，您可能需要从索尼开发者网站获取解锁码。在这种情况下，命令将是： 

      fastboot oem unlock 0x您的解锁码

      您需要将 您的解锁码 替换为您从索尼收到的代码。请查阅您的特定设备的官方索尼文档。

  10. 按照屏幕上的指示操作:

    • 您的手机现在应该会显示有关如何确认bootloader解锁的说明。使用音量和电源按钮进行导航和确认。

  11. 重启您的设备:

    • bootloader解锁后，您的设备可能会自动重启。如果不是，您可以在命令提示符/终端中使用命令 fastboot reboot。

重要提示:

  • 确切的步骤和命令可能因您的特定Xperia设备型号和Android版本而略有不同。

  • 始终参考索尼开发者官方网站以获取您设备的最新和最准确的说明: https://developer.sony.com/open-source/aosp-on-xperia-open-devices/get-started/unlock-bootloader/

通过继续执行这些步骤，您承认所涉及的风险，并对可能出现的任何问题负责。
